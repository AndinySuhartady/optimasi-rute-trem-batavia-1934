# Optimasi Rute Trem Batavia 1934

## Deskripsi Proyek

Proyek ini merupakan implementasi dan analisis berbagai algoritma pencarian lintasan untuk menentukan rute optimal pada jaringan trem Batavia tahun 1934.

## Algoritma yang Dianalisis

* Uniform Cost Search (UCS)
* Dijkstra
* Greedy Best First Search (GBFS)
* A* Search
* Branch and Bound

## File Utama

* `optimasi-rute-trem-batavia-1934.py`

## Tujuan

* Menganalisis strategi algoritma pencarian.
* Membandingkan efisiensi algoritma dalam menemukan rute optimal.
* Mengkaji kelebihan dan kekurangan masing-masing algoritma.

## Teknologi

* Python 3
* Graph Search Algorithms

## Cara Menjalankan

```bash
python optimasi-rute-trem-batavia-1934.py
```
